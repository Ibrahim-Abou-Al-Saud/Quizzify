# Quizzify Client

Quizzify is a comprehensive online quiz and test management platform built with modern Angular. It provides role-based access for Administrators and Users, allowing for seamless test creation, participation, and result tracking.

## Core Features
- **Authentication & Authorization**: Secure login and signup functionality.
- **Admin Portal**: Create and manage tests, add questions, and view overall test results.
- **User Portal**: Take tests interactively and view personal test results and performance.
- **Scalable Architecture**: Utilizing core services, feature modules (Admin/User), and shared UI components (Dashboard, Grids, Spinners).
